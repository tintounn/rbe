----------------------------------------------------------------------------------------------------
 * Introduction
----------------------------------------------------------------------------------------------------

Polymer est une bibliothèque JavaScript open-source permettant de construire des applications web 
à partir de web components. La bibliothèque est développeur par les développeurs et contributeurs 
Google sur GitHub. La bibliothèque est les composant sont tous basé sur les principes de 
conception de Google Material Design.

Polymer est utilisé par un certain nombre de services et de sites web Google,
comme YouTube Gaming, les sites Web Google I/O, Google Play Music 
et Google Sites.


----------------------------------------------------------------------------------------------------
 * Commencer avec polymer
----------------------------------------------------------------------------------------------------

- Installer Node.js (https://nodejs.org/en/)
- Installer Git (https://git-for-windows.github.io/)
- npm install -g bower (Bower)
- npm install -g polymer-cli (Polymer Cli)
- mkdir my-app && cd my-app (Création du dossier de l'application)
- polymer init

A la suite de la commande on demande quel template utiliser, parmis les suivants:

  1) element - A blank element template
  2) application - A blank application template
  3) shop - The "Shop" Progressive Web App demo
  4) starter-kit - A starter application template, with navigation and "PRPL pattern" loading
